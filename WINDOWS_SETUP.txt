NeighborSRT Local Web - Windows Setup

This package can run without Codex if you only want the local web app.
Codex is only needed if you want to install the included Codex skill.

Fast path:

  Double-click:
    install_all_windows.bat

  It installs Python, ffmpeg, openai-whisper, pythainlp, and the Codex skill.
  It can also download Whisper large-v3 if you choose yes.

Manual path:

1. Install Python 3

   Recommended:
     winget install Python.Python.3.12

   Then open a new PowerShell and check:
     py -3 --version

2. Install ffmpeg

   Recommended:
     winget install Gyan.FFmpeg

   Then open a new PowerShell and check:
     ffmpeg -version

3. Install Python packages

   py -3 -m pip install -U openai-whisper pythainlp

4. Start the local web app

   Double-click:
     start_windows.bat

   Or run:
     py -3 app\server.py

   Open:
     http://127.0.0.1:8765

5. Install as a Codex skill, optional

   Double-click:
     install_skill_windows.bat

   This copies the skill to:
     %USERPROFILE%\.codex\skills\neighborsrt

Whisper model note:

The first transcription may download a model such as large-v3. This can be several GB.
For offline use, copy model files into:
  %USERPROFILE%\.cache\whisper

Common models:
  small.pt
  medium.pt
  large-v3.pt
